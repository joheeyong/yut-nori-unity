# 윷놀이 (YutNori) - Unity

한국 전통 보드게임 윷놀이의 Unity 버전입니다.

## 지원 플랫폼
- Android
- iOS
- WebGL (브라우저)
- Windows / macOS / Linux

## 개발 환경
- Unity 2022.3.25f1 (LTS)
- C#

## 시작하기

1. [Unity Hub](https://unity.com/unity-hub) 설치
2. Unity 2022.3.25f1 LTS 설치
3. 이 프로젝트를 Unity Hub에서 열기
4. `Assets/Scenes/MainScene.unity` 실행

## 프로젝트 구조

```
Assets/
  Scenes/      # 게임 씬
  Scripts/     # C# 게임 로직
  Prefabs/     # 재사용 가능한 오브젝트
  Materials/   # 머티리얼
  Textures/    # 텍스처 이미지
  Audio/       # 사운드
```
